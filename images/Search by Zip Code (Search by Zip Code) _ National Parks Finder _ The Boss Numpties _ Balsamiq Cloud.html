What is UI Wireframing ?

Architects sketch out their ideas for buildings before construction starts. They'd be foolish not to!

User Interface designers do the same for their web pages, app screens, or software states.

Wireframing is how you first put your ideas to paper, so you can clarify them, iterate, and pitch them to all stake-holders. This key step saves you time and money by helping you iterate when it's cheapest.
